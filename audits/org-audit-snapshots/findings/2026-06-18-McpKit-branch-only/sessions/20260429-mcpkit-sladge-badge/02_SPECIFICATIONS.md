# Specifications

## Acceptance Criteria

- Add the sladge badge near the README badge header.
- Do not modify package manifests, protocol implementation, or generated SDK
  material.
- Preserve canonical checkout changes by working only in the isolated worktree.
- Record the work in session docs.

## Out of Scope

- Implementing MCP functionality.
- Reconciling the existing ADR, PRD, plan, or research work.
- Merging the prepared badge commit into the dirty canonical checkout.
