# Research

## Repo Fit

McpKit is in scope for the sladge rollout because its docs define it as a
multi-language Model Context Protocol toolkit for standardized AI-to-tool
communication. The planning and specification docs cover AI assistants, MCP
servers, tools, resources, prompts, and agent-framework adapters.

## Local State

Canonical `McpKit` had unrelated local edits in package manifests and Rust
crate manifests, plus untracked ADR, PRD, plan, reference, research, and
`rust/agentora` work. The badge change was prepared in an isolated worktree to
avoid mixing those changes.

## Decision

Treat this as a documentation/governance badge update only. Do not modify MCP
protocol code, package manifests, generated SDK material, or workstream docs.
