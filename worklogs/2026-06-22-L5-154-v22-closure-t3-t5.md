## Pattern: Heredoc + Python is the only reliable mutation path

The shared-lock-DAG problem (26+ concurrent subagent sessions on the
monorepo) caused every `patch` / `write` / `sed -i` tool call to revert
files mid-session. Heredoc + Python heredoc is the only pattern that
persists file changes in this environment. This affects 100% of
mutations in this codebase when multiple parallel agents are active.
