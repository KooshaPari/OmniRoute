## Convention

1. **`#![deny(missing_docs)]` on all lib roots** (already adopted in v14 T8 for 8 core crates; extend to remaining 6).
2. **`#![warn(rust_2018_idioms)]`** in lib roots.
3. **No `unwrap()` in production code** (use `?` + `thiserror` / `anyhow`).
4. **Sealed traits** for extension points (e.g., `pub trait Adapter: sealed::Sealed {}`).
5. **Newtype wrappers** for domain primitives (e.g., `struct UserId(Uuid)` not `String`).
6. **Enums over bool flags** for any >2 state.
7. **`#[must_use]` on all Result/Option returns** that should be checked.
