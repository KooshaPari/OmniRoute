## Test infrastructure

A new `tests/e2e/` directory in each relevant crate. The CI workflow (`.github/workflows/e2e-tests.yml`) runs all `tests/e2e/**/*.rs` on a release runner with a 10-minute timeout per test.

```yaml
name: e2e-tests
on:
  workflow_dispatch:
  schedule:
    - cron: '0 4 * * *'  # 04:00 UTC nightly

jobs:
  e2e:
    runs-on: ubuntu-latest-4core  # release runner
    steps:
      - uses: actions/checkout@v4
      - uses: dtolnay/rust-toolchain@stable
      - run: cargo test --release --test e2e_ -- --test-threads=1
        timeout-minutes: 10
      - uses: actions/upload-artifact@v4
        if: always()
        with:
          name: e2e-traces-${{ github.run_id }}
          path: tests/e2e/traces/
```
