## Format

```rust
// locale.rs
pub enum Locale { En, Es, Ja }

pub fn localize(key: &str, locale: Locale) -> &'static str {
    match (key, locale) {
        ("flag.parse.error", Locale::En) => "invalid flag syntax: {0}",
        ("flag.parse.error", Locale::Es) => "sintaxis de flag inválida: {0}",
        ("flag.parse.error", Locale::Ja) => "フラグの構文が無効です: {0}",
        // ...
    }
}
```

## Adoption gate

- CLI tools detect `LANG` env var
- Default to `en`
- `--locale=es|ja` flag for explicit override
- Error messages route through `localize()`
