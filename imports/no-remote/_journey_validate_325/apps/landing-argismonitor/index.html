Why ArgisMonitor

160+ providers, one API

OpenAI, Anthropic, Gemini, Bedrock, Vertex, Mistral, OpenRouter, Kiro, Cursor, Qoder, Antigravity — all behind a single OpenAI-compatible endpoint.

RTK + Caveman compression

Dramatic token reduction with no quality loss for tool-use payloads. See docs/compression/.

AI-DD / HITL-less

Designed from the ground up for AI-driven development without a human-in-the-loop review gate. See FORK.md § 2.

MCP / A2A native

Built-in Model Context Protocol server, Agent-to-Agent messaging, and OpenAI-compatible HTTP transport. Interop with Claude Code, Codex, OpenCode.

Multi-surface delivery

CLI, Electron desktop, Tauri desktop, PWA, SvelteKit dashboard, MCP server — same code, every target.

Phenotype. integration

Lives in the Argis & Helios family. phenotype.space hosts the docs; pheno.studio hosts the dev/internal surface.
