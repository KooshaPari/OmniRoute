## Intent Statement

vibeproxy-monitoring-unified is the governance + specification home for shared VibeProxy monitoring assets — health checks, alert rules, Prometheus/Grafana surfaces, and probe-latency tracking — across all VibeProxy services. It exists so every VibeProxy deployment consumes the same monitoring configuration (liveness/readiness/startup semantics, error-budget recording rules) and so platform-level alerts are owned once instead of forked per service. Success = a VibeProxy service can `git clone` this repo, drop in the published dashboards + alert rules, and have consistent SLA reporting without authoring its own probes.
