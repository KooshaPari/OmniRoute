## Intent Statement

Paginary federates the four Phenotype documentation pillars — Handbook, Specs, X-driven Dev, and Journeys — into a single VitePress multi-app hub with hot-reload and a shared theme. It exists so contributors can author content in their canonical source repos (PhenoHandbook, PhenoSpecs, phenoXdd, phenotype-journeys) and have it surface uniformly without copy-paste drift. Success = a single `bun run build` produces 4 sub-sites + theme with no manual sync step and no unique source-of-truth living inside Paginary.
