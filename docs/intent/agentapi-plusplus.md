## Intent Statement

agentapi-plusplus is a Phenotype org fork of Anthropic's `coder/agentapi` — an HTTP API gateway that drives 11 different AI-coding-agent CLIs (Claude Code, Cursor, Aider, Goose, Codex, Gemini, Copilot, Amp, Auggie, Opencode, Amazon Q) over an in-memory PTY. It exists so any language/platform can programatically talk to a coding agent without spawning the CLI directly, and the Phenotype fork adds message-formatting, SSE event streams, and 74+ commits of platform hardening. Success = a downstream MCP server, a CI runner, or a fleet dispatcher can `POST /message` to any of the 11 supported agents and get back a structured session.
