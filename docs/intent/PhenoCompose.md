## Intent Statement

PhenoCompose is the canonical Rust driver + FFI layer for the NVMS multi-tier isolation stack (WASM / gVisor / Firecracker) used to deploy and orchestrate Phenotype workloads across macOS, Linux, and Windows. It exists to give Rust callers a single, ergonomic `pheno-compose` API that picks the right isolation tier based on the trust/performance tradeoff of the workload. Success = a deploy target is described once, and PhenoCompose chooses WASM (~1ms), gVisor (~90ms), or Firecracker (~125ms) without the caller re-architecting.
