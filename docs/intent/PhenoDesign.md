## Intent Statement

PhenoDesign (published as `@kooshapari/design`) is the Phenotype design-token + CSS-component package: a framework-agnostic core (color tokens, fonts, W3C DTCG JSON) plus a VitePress theme drop-in that applies the dark-first impeccable baseline to any doc site. It exists so every Phenotype-owned site (specs, handbook, journeys, xDD) shares a single source of color, typography, and component shape without restyling. Success = a new Phenotype VitePress site imports one CSS file and looks on-brand with zero per-site overrides.
