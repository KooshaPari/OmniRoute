# Intent - SessionLedger

## Intent Statement

SessionLedger is the Phenotype-org **OKF-native session compiler**: capture, archive, distill, and replay AI agent sessions into injectable continuation bundles (Acceptance / Contract / Context / Intent). It ships `sl-daemon` (HTTP ingest/search/replay/metrics) and `sl-viewer` (Dioxus desktop/web UI).

## Non-goals

- Not an LLM cost/usage ledger (that is **Tokn** / tokenledger).
- Not a hardware inventory ledger (that is **hwLedger**).
- Not a general PM/spec engine (AgilePlus / Tracera).
